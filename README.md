## 卒業制作ポートフォリオ

### サービス概要
旅先で出会ったお土産を記録しながら、新しいお土産との出会いを発見できるSNS型レビュー&お土産マッチングサービスです。<br>
お土産のレビューを見たり、好みに合ったレコメンドや国別のランキングを通して、新しいお土産を見つけることができます。<br>
また、お土産の記録と同時に、旅行の写真や日記を追加できるので、コレクションを楽しみながら、旅行の思い出の記録としても利用可能です。
<br><br>

### このサービスへの思い・作りたい理由
このサービスでは、「お土産との出会いは一期一会である」という考えのもと、お土産との「出会いの発見」と「思い出の記録」に注目し、旅人が新しいお土産を発見したり、お土産を購入した時の楽しかった記憶を思い返したりできるようなサービスを考えました。

#### 1. お土産との出会いの発見
お土産といえば、有名なお菓子やその地域の特産品を思い浮かべるかもしれませんが、世界中のあらゆる商品がお土産となりうると考えています。<br>
そんな無数のお土産のなかで、どこでどんな出会いがあるかわかりません。<br>
旅先での時間は限られているため、もちろん気に入ったお土産に出会えないこともあります。<br>
私の場合は、後から「こんないいお土産があったのか...」「これ見つからなかったけど、このお店にあったのか...」というように、帰宅後に後悔することも少なくありませんでした。

反対に、とてもいいお土産なのにあまり知られていないようなものだったり、ご当地限定なのに特に気にせずスルーされてしまっていたりというような、もっと注目されるべき隠れたお土産たちもいるのではないかと考えています。

「せっかくの素敵なお土産は、多くの人の手に取ってもらいたい。」<br>
そんな思いから、お土産をシェアできるようなサービスを作ることで、これから旅に出る人がお土産との出会いを新たに発見できる仕組みを作りたいと考えました。<br>
知られざる隠れたお土産たちにスポットライトを当てることで、最終的には、地域活性化の一助になるようなそんなサービスを目指したいと考えています。

#### 2. お土産との思い出の記録
お土産を買う理由は人それぞれだと思いますが、私の場合は、そのお土産を見ると旅行をしていた時の楽しかった思い出や光景が思い浮かぶことが大きな理由の一つになっています。<br>
しかしながら、お菓子などは食べたらなくなってしまいますし、引越しなどで昔に買ったお土産を捨てなくてはならないという場合もあると思います。<br>
さらには、時間が経てば、その楽しかった思い出も徐々に薄れていってしまいます。

「せっかく出会ったお土産との記憶をずっと残しておきたい」<br>
そういった思いから、お土産の購入に至ったきっかけや旅先でのエピソードを、お土産と共に記録できるような仕組みにもしたいと考えました。<br>
このサービスに記録しておけば、形はなくなったとしても、購入したお土産たちは旅行の思い出と共に存在し続けます。
<br><br>

### ユーザー層について
- お土産をよく買う人
- 旅行計画中の人で、どんなお土産を買えば良いか分からない人
- 買ったお土産を記録しておきたい人
- 旅行の思い出を記録しておきたい人
<br><br>

### サービスの利用イメージ
#### ①旅行前
- #### 利用イメージ
  - カテゴリーや旅行先、おすすめ度で絞り込んでお土産を探す
  - 投稿されたお土産を欲しいものリストに保存して後で見返す
  - 国別のランキングや人気投稿からお土産を見つける
  - 投稿に対して質問をする
  - 事前に任意のアンケートに答えることで、回答に沿ったお土産が毎日何個かランダムでレコメンドされ、気になるものを保存する
  - 特定のお土産がどこで購入できるのかを投稿に紐づけられた位置情報から知る（Mapから探すことも可能）
  - 特定のお土産がいくらくらいで購入できるのかを投稿に紐づけられた金額情報から知る

- #### メリット
  - ガイドブックには載っていないお土産を発見できる
  - 旅行先のお土産の購入場所がわかるので、計画を立てやすくなる（旅先で効率的に行動できる）
  - 自分の好みに合わせたお土産を探すことができる
  - 具体的な質問をすることでリアルタイムに会話しながらより詳しい情報を得られる

#### ②旅行後
- 利用イメージ
  - 購入したお土産を検索し、それを購入したお店、値段、おすすめ度、レビューなどと共に記録（投稿）する
  - 検索にヒットしないお土産は、お土産の名前と説明、写真、カテゴリーと共に新規登録する
  - 購入のきっかけや購入した当時の旅先の写真も同時に記録する（公開はされない）
  - お土産の詳細ページ情報を別のSNSへの投稿やURLコピーをして、友人や家族にシェアする

- メリット
  - お菓子など形がなくなってしまうお土産も含め、旅の戦利品のようにどんどんコレクションを楽しめる
  - 思い出の記録ができる
  - 友人や家族にお土産をおすすめができる
<br><br>

### サービスの差別化ポイント・推しポイント
- お土産に特化<br>
もちろん他のSNSや旅行ガイドブックでもお土産を探すことは可能ですが、情報量が非常に多く、特定のジャンルに絞って検索したい場合は探しづらいと考えています。
このサービスでは、お土産に特化しているため、知りたいお土産の情報を最短距離で探すことができます。

- 新しいお土産に出会える<br>
たくさんあるお土産の中から自分の好みにあったお土産を表示してくれるレコメンド機能や人気投稿やランキング機能で、これまで知らなかったお土産を発見するきっかけを作ります。
まだガイドブックにも載っていないようなニッチなお土産やまだ話題になっていない最新トレンドのお土産にも出会えるかもしれません。

- 思い出と共にコレクション<br>
単純にお土産をコレクションする楽しさに加え、「お土産を見ると（もしくは食べるとき）旅行をしていた時の楽しかった思い出や光景が思い浮かぶ」という経験から、お土産の記録とともに、旅行の写真や日記を追加できるような仕組みを作りました。<br>
形がなってしまうお土産であっても、このサービスに記録しておけば、旅行の思い出と共に残り続けます。
<br><br>

### 機能候補
#### MVP
- 会員登録
  - Email
  - パスワード
  - ユーザー名、アイコンは後から変更可能
- 投稿機能
  - お土産名（お土産名を検索でき、ない場合は新規追加が可能）
  - 位置情報（施設名を検索できる）
  - 金額（多通貨に対応）
  - 写真
  - おすすめ度
  - レビュー
  - シチュエーション（自分用、家族用、友人用、会社の同僚用など）
  - 使う人の年齢層
  - 旅行の記録（写真、テキストなど、外部公開はされない想定）
- お土産の新規登録
  - 写真
  - 説明
  - カテゴリー
- お土産詳細ページ
  - お土産に紐づけられた投稿一覧を表示
  - 並び替え（最新順、金額順）
  - 絞り込み（国名、施設名）
  - 関連土産へのリンク（カテゴリーや国が同じもの）
- タイムライン
  - 投稿の一覧表示
  - 絞り込み機能（カテゴリー、国名）
  - 特定のお土産をほしい物リストに追加できる
  - フォロー機能
  - 質問機能（コメント）
- お土産検索
  - お土産名、カテゴリー、国名から、検索可能
- マップ
  - 投稿に紐づけられた位置情報からお土産を表示する
  - 一部お店の情報も掲載（詳しい情報はGoogle Mapへのリンク）

#### 本リリース
- お土産好みアンケート（回答は任意で、後から変更可能）
  - アンケートの回答に応じてランダムにレコメンドする
  - スワイプで気になるものを保存できる
- 人気投稿順
  - 閲覧数に応じて、人気投稿が上に表示される
  - タイムラインとタブを分けて表示
- 新規登録されたお土産の運営によるチェック
  - 画像の著作権のチェック
  - 情報の正確性を担保するため
  - 登録されたら運営に通知
  - 承認されるまでは未承認タグがつくなどユーザーが混乱しないように情報を提供する

#### 将来的な拡張
- 多言語対応（インバウンドの旅行客も使用できるように）
- アップロードする画像の加工機能
- お菓子などの食品に関しては、賞味期限を登録でき、近づいてくると通知してくれる機能
- 動画による投稿（保存容量や費用的な面を要検討）
- ネイティブアプリの実装
- 閲覧データから自動でおすすめのお土産がレコメンドされる

#### ビジネス化した場合（アイデアベース）
- お土産データ化
  - 年齢別、シチュエーション別におすすめのお土産を提案、ランキング
  - マーケティングのデータとして活用できる
- お店側からのPR投稿やおすすめお土産の特集記事を掲載
- お店側との連携により最新の在庫情報を確認できる
- 海外土産系のECサイトとの提携で日本にいてもお土産を購入が可能にする
- 会員ランクとポイント制度
  - 次回の旅行で使えるお土産クーポンを進呈（再度訪れるきっかけを提供し、地域活性化につなげる）
  - ランクに応じてお土産をプレゼント
<br><br>

### 機能の実装方針予定
- バックエンド
  - Ruby on Rails（APIモード）

  - React（Next.js）

  - 多通貨での投稿機能：Open Exchange Rates API
